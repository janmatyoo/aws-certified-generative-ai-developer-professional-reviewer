### AI Agents

**What it is:**  
Fully managed AWS service that enables developers to create, deploy, and manage autonomous, AI-driven agents capable of executing multi-step business tasks

Uses the reasoning of foundation models (FMs), APIs, and data to break down user requests, gathers relevant information, and efficiently completes tasks


**Used for:**
- has a default pre-processing prompt that can determine if user input is safe to use
- enable default pre-processing prompts in Agents to add an additional layer of security
- pre-processing prompt will evaluate user input before the input reaches the FM
- helps identify and filter potentially malicious prompts
